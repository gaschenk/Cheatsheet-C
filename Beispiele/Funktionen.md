# Beispiele für Funktionen
```c
// Funktionsname: "example1"
// Rückgabe-Datentyp: void (keine Werte, nichts spezifiziert)
// Parameter: keine
void example1() {
}

// Funktionsname: "example2"
// Rückgabe-Datentyp: int (Integer)
// Parameter: keine
int example2() {
	return 0;
}

// Funktionsname: "example3"
// Rückgabe-Datentyp: void (keine Werte, nichts spezifiziert)
// Parameter: int (Integer) als "value" definiert
void example3(int value) {
}

// Funktionsname: "example4"
// Rückgabe-Datentyp: int (Integer)
// Parameter: int (Integer) als "returnMe" definiert
void example4(int returnMe) {
	return returnMe;
}

// Funktionsname: "add"
// Rückgabe-Datentyp: int (Integer)
// Parameter: int (Integer) a | int (Integer) b
int add(int a, int b){
	return a+b;
}
```
